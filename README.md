# Map & Location Testing App - تجربة أدوات الخرائط والنماذج 📍

مشروع تقني مبني باستخدام **Flutter** يهدف إلى اختبار وتجربة أدوات الخرائط (Maps SDK) ومعالجة نماذج البيانات (Models). يُعد هذا المشروع مختبراً برمجياً (Sandbox) لاستكشاف كيفية دمج الخرائط التفاعلية والتعامل مع الإحداثيات الجغرافية وتطوير منطق برمجي مرن لنماذج البيانات.

## 🎯 الهدف من المشروع

اختبار تكامل خرائط Google أو الخرائط مفتوحة المصدر داخل تطبيقات فلاتر، وتجربة تحويل البيانات الجغرافية الخام إلى نماذج (Models) قابلة للاستخدام داخل التطبيق، مع التركيز على دقة تحديد المواقع وإدارة الحالات.

## ✨ المميزات والوظائف (Features)

### 1. دمج الخرائط التفاعلية (Map Integration)

* **عرض الخرائط:** عرض الخريطة التفاعلية مع إمكانية التحكم في التكبير والتصغير (Zoom) والتحريك.
* **تحديد الموقع (Geolocation):** اختبار الحصول على موقع المستخدم الحالي وإظهاره على الخريطة.
* **العلامات (Markers):** إضافة وتخصيص العلامات على الخريطة للإشارة إلى نقاط اهتمام معينة.

### 2. هندسة النماذج (Model Architecture)

* **تحويل البيانات (Data Parsing):** تجربة تحويل استجابات JSON الخاصة بالخرائط إلى كائنات (Objects) برمجية.
* **هيكلية البيانات:** بناء نماذج بيانات مرنة تتعامل مع الإحداثيات (Latitude & Longitude) والعناوين النصية.

### 3. تجربة الواجهة (UI Experiments)

* **تراكب العناصر (Map Overlays):** وضع عناصر واجهة مستخدم فوق الخريطة بشكل انسيابي.
* **البحث المكاني:** تجربة منطق البحث عن المواقع والحصول على اقتراحات العناوين.

## 🛠 التقنيات المستخدمة (Technical Stack)

* **Flutter & Dart:** المنصة الأساسية للتطوير.
* **Google Maps Flutter / Leaflet:** أدوات عرض الخرائط.
* **Geolocator & Geocoding:** مكتبات التعامل مع نظام تحديد المواقع العالمي وتحويل الإحداثيات إلى نصوص والعكس.
* **Custom Models:** بناء نماذج بيانات مخصصة لتمثيل المواقع الجغرافية.

## 📂 هيكلية المشروع (Project Structure)

```text
lib/
├── models/        # نماذج بيانات المواقع والإحداثيات (Location Model)
├── screens/       # واجهة الخريطة وشاشات الاختبار
├── services/      # خدمات جلب الموقع الجغرافي (Location Services)
├── widgets/       # عناصر الواجهة المخصصة فوق الخريطة
└── utils/         # أدوات مساعدة للتعامل مع الإحداثيات والخرائط

```

## 🚀 كيفية التشغيل (Installation & Setup)

1. **استنساخ المستودع:**
```bash
git clone https://github.com/salem-baqatyan/map_app.git

```


2. **تحميل المكتبات:**
```bash
flutter pub get

```


3. **ملاحظة هامة:** يتطلب تشغيل الخرائط إضافة مفتاح واجهة برمجة التطبيقات (API Key) في ملفات الإعداد الخاصة بـ Android و iOS.
4. **تشغيل المشروع:**
```bash
flutter run

```



## 📝 الدروس المستفادة (Learning Outcomes)

* فهم كيفية التعامل مع دورة حياة الخرائط (Map Lifecycle) داخل تطبيقات الجوال.
* إدارة الأذونات (Permissions) الخاصة بالموقع الجغرافي على نظامي Android و iOS.
* تحسين أداء التطبيق عند ريندر (Rendering) الخرائط واستهلاك البيانات.
